These files were sent by Karen Felzer in a series of emails on July 26.

I added .txt to some of these files.

Descriptions:

	RatesOriginal has M>=5 earthquake rates, NSHMP02 method

	RatesRecommended has M>=5 earthquake rates
	using the recommended methods

	latloc: latitude location of each point in the data sets above

	lonloc: Longitude locations of each point in the data sets above

SeismicityRateDoc.* is her write up.

"_v1" was added to the file names to allow versioning.
