Title: Field_2000_AttenRel

Description: This implements the Attenuation Relationship developed by Field (2000, Bulletin of the Seismological Society of America, vol 90, num 6b, pp S209-S221)

Supported Intensity-Measure Parameters:

  • pgaParam - Peak Ground Acceleration
  • saParam - Response Spectral Acceleration
  • saParam - Peak Ground Velocity (from 1-sec SA using the Newmark & Hall (1982) amp factor of 37.27*2.54/100)

Other Independent Parameters:

  • magParam - moment Magnitude
  • distanceJBParam - closest distance to surface projection of fault
  • vs30Param - Average 30-meter shear-wave velocity at the site
  • basinDepthParam - Depth to 2.5 km/sec S-wave velocity
  • fltTypeParam - Style of faulting
  • componentParam - Component of shaking (only one)
  • stdDevTypeParam - The type of standard deviation

updateCoefficients

protected void updateCoefficients()
This function determines which set of coefficients in the HashMap are to be used given the current intensityMeasure (im) Parameter. The lookup is done keyed on the name of the im, plus the period value if im.getName() == "SA" (seperated by "/"). SWR: I choose the name update instead of set, because set is so common to java bean fields, i.e. getters and setters, that set() usually implies passing in a new value to the java bean field. I prefer update or refresh to functions that change internal values internally
